# Sonethavy Yanong Personal Portfolio

A modern multi-page personal portfolio website for Sonethavy Yanong, a 16-year-old creator from Laos who enjoys gaming, technology, learning new skills, and building a future as an entrepreneur.

## Owner

Sonethavy Yanong

## About

I am a 16-year-old creator from Laos.
I enjoy Roblox, technology, and learning new skills.

## My Vision

My goal is to build successful projects and create value for people in Laos.
I want to use technology, creativity, and hard work to create something meaningful for the future.

## Website Identity

- Gamer 🎮
- Technology learner 💻
- Future entrepreneur 🚀

## Design Style

- Dark futuristic theme
- Smooth animations
- Modern young creator portfolio

## Pages

- `index.html` - Home page with hero, profile photo, My Mission, and a future-building framework
- `about.html` - About page with personal story, profile photo, and creator values
- `interests.html` - Gaming, technology, learning, business goals, and learning roadmap
- `dream.html` - Future entrepreneur dream statement with a practical path forward
- `contact.html` - Contact placeholders, social media buttons, and portfolio status note

## Files

- `styles.css` - Shared dark futuristic theme, glassmorphism cards, responsive layout, and animations
- `script.js` - Loading animation, scroll fade-in effects, active navigation, and page transitions
- `photo.jpg` - Profile photo used on the Home and About pages

## How to View

Open `index.html` in a web browser.

## Technical Style

The website uses a dark futuristic gradient, soft animated background movement, glassmorphism cards, smooth hover effects, fade-in scroll animations, responsive grids, roadmap sections, and a mobile-friendly layout.
